# UI 规范索引

> 目标态规范(尚未落地)。

本目录定义 Loom 前端 redesign 后的视觉设计系统与组件规范。

## 文档列表

- [设计系统](design-system.md) — 色彩、字体、圆角、阴影、动效 token 定义
- [组件规范](components.md) — 按钮、卡片、toast、chip、modal、tabs、nav 等组件设计规范

## 设计语言

redesign 后的目标设计语言:混合风格 — 暗色基底 + emerald 主色 + shadcn/ui 三级按钮变体体系。保留绿色品牌基因,采用 sans-serif 为主字体,等宽字体仅用于代码/路径/数据值。

设计决策记录见 [spec](../superpowers/specs/2026-07-02-ui-style-redesign.md)。
